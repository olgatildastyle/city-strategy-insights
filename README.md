# Стратегії та рішення громад
Аналітичний вебзастосунок для аналізу рішень місцевих рад, стратегій розвитку, досягнень громад та порівняння напрямків їхнього розвитку.

## Про проєкт
Мета застосунку — зробити дані про громади більш зрозумілими та доступними.  
Користувач може переглядати громади, аналізувати рішення місцевих рад, бачити пов’язані досягнення та порівнювати громади між собою.

Проєкт створено як MVP civic-tech дашборду.

Дані
У поточній версії використовуються mock-дані, які імітують реальні рішення місцевих рад, стратегії розвитку та досягнення громад.
У майбутньому застосунок можна розширити підключенням реальних відкритих даних з офіційних джерел.

Статус
MVP у розробці.
